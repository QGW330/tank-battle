# 🚀 快速部署指南

## 最简单的方法：Netlify Drop（无需注册）

### 步骤：

1. **访问 Netlify Drop**
   - 打开浏览器，访问：https://app.netlify.com/drop

2. **拖拽文件夹**
   - 直接将整个 `坦克大战` 文件夹拖拽到网页上
   - 或者点击选择文件夹

3. **获取链接**
   - 等待几秒钟，Netlify 会自动生成一个链接
   - 链接格式类似：`https://随机名称.netlify.app`
   - 这个链接可以分享给任何人！

4. **完成！**
   - 现在任何人都可以通过这个链接访问你的游戏了
   - 链接永久有效（只要不删除）

---

## 方法二：GitHub Pages（需要 GitHub 账号）

### 步骤：

1. **创建 GitHub 仓库**
   - 访问 https://github.com
   - 点击右上角 "+" → "New repository"
   - 输入仓库名称（如：tank-battle）
   - 选择 Public
   - 点击 "Create repository"

2. **上传文件**
   - 在仓库页面，点击 "uploading an existing file"
   - 将所有文件（index.html, style.css, game.js）拖拽上传
   - 点击 "Commit changes"

3. **启用 GitHub Pages**
   - 点击仓库的 "Settings"
   - 左侧菜单找到 "Pages"
   - 在 "Source" 下选择 "main" 分支
   - 点击 "Save"

4. **获取链接**
   - 等待1-2分钟
   - 访问：`https://你的用户名.github.io/仓库名`
   - 例如：`https://zhangsan.github.io/tank-battle`

## 方法三：Vercel（推荐，功能强大）

### 步骤：

1. **访问 Vercel**
   - 打开 https://vercel.com
   - 使用 GitHub 账号登录（或注册）

2. **创建项目**
   - 点击 "Add New..." → "Project"
   - 选择 "Import Git Repository" 或直接拖拽文件夹

3. **部署**
   - 如果是拖拽：直接上传文件夹
   - 如果是 Git：选择仓库后点击 "Deploy"
   - 几秒钟后获得链接

---

## 方法四：Surge.sh（命令行）

如果你熟悉命令行：

```bash
# 安装 surge（需要 Node.js）
npm install -g surge

# 进入项目目录
cd 坦克大战

# 部署
surge

# 按提示输入邮箱和密码（首次使用需要注册）
# 输入域名（或使用随机域名）
```

## 遇到问题？

- **页面显示空白**：检查文件路径是否正确
- **样式丢失**：确保所有文件都在同一目录
- **游戏无法运行**：打开浏览器控制台查看错误信息
