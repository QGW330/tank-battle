# 坦克大战游戏

一个基于 HTML5 Canvas 的坦克大战网页游戏。

## 游戏特性

- 🎮 流畅的坦克控制
- 🎯 智能敌人AI
- 💥 爆炸特效
- 🌿 精美的平原草原背景
- 🎨 写实的坦克设计

## 控制说明

- **移动**: WASD 或方向键
- **射击**: 空格键或鼠标左键
- **瞄准**: 鼠标移动

## 部署到网络

### 方法一：Netlify Drop（最简单，推荐）

1. 访问 [Netlify Drop](https://app.netlify.com/drop)
2. 将整个项目文件夹拖拽到页面上
3. 等待几秒钟，Netlify 会自动生成一个链接
4. 分享链接给朋友即可！

### 方法二：GitHub Pages

1. 在 GitHub 上创建一个新仓库
2. 将所有文件上传到仓库
3. 在仓库设置中启用 GitHub Pages
4. 选择 main 分支作为源
5. 访问 `https://你的用户名.github.io/仓库名` 即可

### 方法三：Vercel

1. 访问 [Vercel](https://vercel.com)
2. 注册/登录账号
3. 点击 "New Project"
4. 导入你的项目文件夹
5. 点击部署，几秒钟后获得链接

### 方法四：其他静态托管服务

- **Surge.sh**: `surge` 命令部署
- **Firebase Hosting**: Google 的免费托管服务
- **Cloudflare Pages**: 快速且免费

## 本地运行

直接用浏览器打开 `index.html` 文件即可。
